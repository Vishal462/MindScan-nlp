# MindScan

MindScan is a Natural Language Processing (NLP) project built with Streamlit.  
It demonstrates advanced text analysis and model deployment in a clean, interactive web app.

**Live Demo:** [Run MindScan here](https://mindscan-nlp.streamlit.app/)
---
## Features
- Interactive Streamlit interface
- NLP model inference with safetensors
- Real‑time text analysis and visualization
- Lightweight deployment on Streamlit Community Cloud

### 1. Clone the repository
```bash
git clone https://github.com/your-username/mindscan-nlp.git
cd mindscan-nlp
```
### 2. Install dependencies
```bash
pip install -r requirements.txt
```
### 3. Run locally
```bash
streamlit run project.py
```
